Game Class Version 1.0

Version Notes:
----------------
This "class" tells the computer to select a number between 1 and 100
and then asks the player to guess the number.  If incorrect it will
allow them to guess again until the correct number is selected.

To Use:
-----------------
*New Issues (https://github.com/andrewhooge/iron_yard_homework/issues)
- Begin by running program in terminal by prompting ruby_game.rb
- Select a number between 1 and 100
- User gets five chances to get the number correct
- Program will let the user know if the number is higher or lower than random number that is generated by the program
